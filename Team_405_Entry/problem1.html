3 Ways to centre a Div inside another Div: 

 By: Sameer Choudhary 2106029 

Method 1:

Using Margin on flex:
Added CSS and used display:flex Property on the outer div and margin:auto Property on the inner div.

Method 2:

Using Table cell:
Added CSS and used display:table-cell Property on the outer div and display:inline-block Property on the inner div.

Method 3:

Using Position:
Added CSS and used position:relative Property on the outer div and position:aboslute Property on the inner div.
