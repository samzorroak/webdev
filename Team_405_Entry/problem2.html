(a.) Difference between let, const & var 

 By: Sameer Choudhary 2106029 

Let:

  • Let is used to declare variables and assign value to them.

  • Scope of a let variable is only block limited.

  • We can update the variable declared by let.

  • But we cannot re-declare the variable defined with let keyword.
                JAVASCRIPT CODE SAMPLE:
                let x = 50;
                document.write(x"); // This will print 50
                x = 100;
                document.write(x"); // This will print 100
                // let x = 500; -->(Not allowed)
                function func() 
                {
                    do{
                    let y = 200;
                        document.write("y"); // This will print 200
                    }while(y > 1000);
                    document.write(y); // This will not print
                }

Const:

  • Const is used to declare Constants with fixed value to them.

  • Const are just like let except they are immutable.

  • Scope of a Const variable is only block limited.

  • We cannot update the constant declared by const.
                JAVASCRIPT CODE SAMPLE:
                const a = 50;
                document.write(x"); // This will print 50
                // a = 500; -->(Not allowed)
                if(true)
                    {
                        document.write(a); // This will print 50
                        const b = 100;
                        document.write(b); // This will print 100
                    }
                    document.write(b); // This will not print

Var:

  • Var is used to declare variables and assign value to them.

  • The scope of the var keyword is the global.

  • We can update the variable declared by var.

  • We can re-declare the variable defined with var keyword.
                JAVASCRIPT CODE SAMPLE:
                var v = 100;
                    document.write(v); // This will print 100
                    var v = 200;

 (b.) double equals (==) & triple equals (===) 

Double equals (==):

  • Double equals (==) is a comparison operator.

  • It transforms the operands having the same type before comparison.

  • It checks whether its two operands are the same or not.

  • If 2 variables have different data type but same value, then returns True.

  • A string with no numeric value is converts to NaN, which returns false.

                JAVASCRIPT CODE SAMPLE:
                var var1 = 100;
                var var2 = "100";
                var var3 = 100;
                document.write(var1==var2);
                document.write(var1==var3);

Output:

triple equals (===):

  • Triple equals (===) is a strict equality comparison operator.

  • Returns false for the values which are not of a similar type.

  • Value are not implicitly converted to some other value before comparison.

  • If the variable values are of different types, then the values are considered as unequal.

  • If we compare 2 with “2” using ===, then it will return a false value.
                JAVASCRIPT CODE SAMPLE:
                var var1 = 100;
                var var2 = "100";
                var var3 = 100;
                document.write(var1===var2);
                document.write(var1===var3);

 (c.) Arrow function & normal function 

Arrow Function:

  • Syntax:
                        var x = (parameters) =>
                        // statements

  • There are no arguments objects available in arrow functions.

  • They are only callable and not constructible.

  • We get Error on trying to construct an arrow functions using the "new" keyword.
                JAVASCRIPT CODE SAMPLE:
                var sum = (v1,v2) => (v1+v2);
                document.write(sum(5,5));

Normal Function:

  • Syntax:
                        var x = name(parameters){
                        //statements
                    };
  • There are arguments objects available in regular functions.

  • They are constructible and callable.

  • They can be called using the "new" keyword.

                JAVASCRIPT CODE SAMPLE:
                var sum = function(v1,v2)
                {
                    return (v1+v2);
                };
                document.write(sum(4,5));
